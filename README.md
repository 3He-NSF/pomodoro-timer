# pomodoro-timer

Simple static pomodoro timer site.

## Live Site

https://3He-NSF.github.io/pomodoro-timer/

## Local Preview

From this directory, start a simple static server:

```bash
python3 -m http.server 4173
```

Then open:

```text
http://localhost:4173
```

## Deploy

This site is intended to be published with GitHub Pages from the `main` branch root.
